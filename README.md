# Telegram-бот «Ежедневный фокус»

Простой MVP-бот помогает выбрать одну главную задачу на день, подготовить шаблон идеи для короткого ролика и открыть обучающий курс в формате Telegram Mini App.

## Возможности

- После `/start` бот показывает постоянную клавиатуру с основными действиями.
- `/start` — объясняет идею бота.
- `/help` — показывает список команд.
- `/plan` — запускает диалог: главная задача и три шага.
- `/focus` — даёт короткий совет по концентрации.
- `/check` — предлагает отметить выполнение задачи кнопками.
- `/content` — спрашивает тему и готовит боль аудитории, обещание, пять хуков и структуру ролика.
- `/app` — показывает кнопку для открытия обучающего Mini App.
- `/cancel` — отменяет текущий незавершённый диалог.

Кнопки дублируют команды и запускают ту же логику:

- `План дня` → `/plan`;
- `Контент` → `/content`;
- `Фокус` → `/focus`;
- `Проверка` → `/check`.

Команды остаются доступны, даже если пользователь предпочитает не использовать клавиатуру.

Планы хранятся только в памяти процесса. Они исчезают после перезапуска бота и автоматически считаются неактуальными на следующий день.

Команда `/content` не использует OpenAI API или другие генеративные сервисы. Ответ собирается локально из заранее подготовленных шаблонов по схеме «боль → обещание → хуки → основная мысль → CTA».

## Требования

- Python 3.10 или новее.
- Токен Telegram-бота, полученный у [@BotFather](https://t.me/BotFather).

## Установка

Откройте терминал в корне проекта и создайте виртуальное окружение:

```powershell
python -m venv .venv
.venv\Scripts\Activate.ps1
```

Для macOS или Linux команда активации выглядит так:

```bash
source .venv/bin/activate
```

Установите зависимости:

```bash
python -m pip install -r requirements.txt
```

## Настройка

Создайте локальный файл `.env` из примера.

В PowerShell:

```powershell
Copy-Item .env.example .env
```

В macOS или Linux:

```bash
cp .env.example .env
```

Откройте `.env` и замените значение на настоящий токен:

```dotenv
TELEGRAM_BOT_TOKEN=1234567890:your_real_token
WEBAPP_URL=https://your-public-site.example
```

Файл `.env` исключён из Git. Не публикуйте токен и не добавляйте его в исходный код.

`WEBAPP_URL` должен указывать на опубликованную страницу `webapp/index.html` по публичному HTTPS-адресу. Если значение не задано или адрес начинается не с `https://`, остальные команды продолжат работать, а `/app` сообщит, что приложение ещё не настроено.

## Mini App

Статические файлы приложения находятся в папке `webapp/`:

- `webapp/index.html` — структура страницы;
- `webapp/style.css` — мобильное оформление;
- `webapp/app.js` — модули, уроки, тексты и навигация.

Для локального просмотра запустите отдельный статический сервер из корня проекта:

```bash
python -m http.server 8080 --directory webapp
```

Затем откройте [http://127.0.0.1:8080](http://127.0.0.1:8080). Локальный HTTP-адрес подходит только для разработки в браузере. Telegram должен получить доступ к приложению по публичному HTTPS-адресу.

Для запуска внутри Telegram опубликуйте содержимое папки `webapp/` на любом статическом HTTPS-хостинге, укажите итоговый адрес в `WEBAPP_URL`, перезапустите бота и отправьте команду `/app`.

## Запуск

```bash
python bot.py
```

После сообщения `Focus bot is starting` откройте бота в Telegram и отправьте `/start`. Бот покажет клавиатуру с четырьмя основными действиями.

Остановить бота можно сочетанием `Ctrl+C`.

## Быстрая проверка кода

```bash
python -m py_compile bot.py
```

Для полной ручной проверки запустите бота и пройдите сценарии:

1. Вызовите `/start` и убедитесь, что появились кнопки `План дня`, `Контент`, `Фокус` и `Проверка`.
2. Нажмите `План дня`, введите задачу и три шага, затем нажмите `Проверка`.
3. Нажмите `Контент`, отправьте тему ролика и убедитесь, что ответ содержит боль аудитории, обещание, пять хуков и структуру с CTA.
4. Нажмите `Фокус` и убедитесь, что бот прислал короткий совет.
5. Повторите основные сценарии командами `/plan`, `/content`, `/focus` и `/check`, чтобы проверить обратную совместимость.
6. Откройте локальную страницу Mini App, перейдите из списка модулей в список уроков, откройте урок и дважды воспользуйтесь кнопкой назад.
7. После публикации по HTTPS вызовите `/app` в Telegram и убедитесь, что кнопка `Открыть обучение` запускает страницу внутри клиента.

## Ограничения MVP

- Базы данных нет.
- План существует только в запущенном процессе.
- Автоматических вечерних напоминаний нет; пользователь сам вызывает `/check`.
- Контент-план строится по универсальным шаблонам и не учитывает внешние данные или контекст аудитории.
- Mini App не хранит прогресс, не содержит оплаты, сложной авторизации или кнопки завершения урока.
- Для открытия Mini App внутри Telegram требуется отдельно опубликовать папку `webapp/` по HTTPS.
